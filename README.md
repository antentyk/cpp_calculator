# cpp_calculator
Calculator project during C++ course at UCU

Current operators:
- binary: <b>+ - / *</b>
- unary: <b>+ -</b>
Current functions:
- binary: <b>min max pow</b>
- unary: <b>sin cos</b>

It supports nested expressions as well

Examples:
    Input: <b>pow(1 + sin(2 * 3), cos(min(1, 2 + 2 / 7)))</b>
    Output: <b>0.837735</b>
